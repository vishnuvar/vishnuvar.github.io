5.    POWER BI Data Gateways:

a.     a network bridge between on-premises data and several Microsoft cloud services like Power BI, PowerApps, Power Automate, Azure Analysis Services, and Azure Logic Apps.

b.    organizations can keep databases and other data sources on their on-premises networks, yet securely use that on-premises data in cloud services.

c.     On-premises data gateway - Power BI | Microsoft Docs

d.    You can install on local computer and so on.

e.     Real time Admin team manages installation

f.      Wherever Database is available the installation is done on the same system.



6.    POWER BI Apps:

a.     A Power app is a Power BI content type that combines related dashboards and reports, all in one place. An app can have one or more dashboards and one or more reports, all bundled together. Apps are created by Power BI designers who distribute and share the apps with their colleagues. There are many different ways a designer can share an app.
